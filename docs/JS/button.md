### button 按钮

概述 #
基本表单组件，支持 input 和 textarea，并在原生控件基础上进行了功能扩展，可以组合使用。

代码示例 #

#### API

##### Input props

|   属性    | 说明          |      类型      | 默认值 |
| :-------: | :------------ | :------------: | :----: |
|   name    | 字体图标 name |     String     |
|   color   | 颜色          |     String     |
|   size    | 大小          | String, Number |
| fontClass | 类名          |     String     |
